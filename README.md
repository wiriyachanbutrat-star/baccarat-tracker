# Baccarat helper (เว็บคำนวณผล)

Simple local web page to record rounds and get a basic suggestion whether to bet `Player` or `Banker` based on historical counts.

Usage:

1. Open `index.html` in your browser.
2. Click the buttons to add a round: `Player`, `Banker`, or `Tie`.
3. The page shows totals, percentages, history, and a simple suggestion.

Files:
- `index.html` - main UI
- `style.css` - styles
- `script.js` - logic
